# DevConnect

## Overview

DevConnect is a full-stack web application that enables developers to connect, collaborate, and build software projects together.

The platform allows developers to create professional profiles, showcase their skills, create collaborative projects, and invite other developers to join their teams.

This project is being developed as a real-world portfolio application using modern Java backend technologies and a modern frontend.

---

## Project Goals

- Build a production-style full-stack application
- Practice clean architecture and REST API development
- Learn real-world software development workflow
- Implement authentication, authorization, project collaboration, and team management features
